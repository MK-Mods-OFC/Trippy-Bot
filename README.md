### Para usuarios de Termux
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/MK-Mods-OFC/Trippy-Bot
> cd Trippy-Bot
> npm install
```
###### Iniciar
```bash
> node index.js
```
